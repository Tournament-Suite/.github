# Contributing to Tournament Suite

Thank you for your interest in Tournament Suite. We are currently in pre-production and our core platform repositories are managed privately. This document outlines how you can get involved.

## Ways to Engage

### Report Issues
Found a bug on [tournamentsuite.com](https://tournamentsuite.com) or the [live demo](https://arena.gamertd.com)? Open an issue using our bug report template and we'll investigate.

### Feature Requests
Have an idea for a tournament format, game integration, or platform feature? Open a feature request — we read every one.

### Developer API & Integrations
We offer a full REST and WebSocket API for building on top of Tournament Suite. See the [Developer Docs](https://tournamentsuite.com/developer) to get started. API feedback and integration questions are welcome as issues in this repository.

### Partnership & White-Label Inquiries
If you are an esports organization, league operator, or platform builder interested in deploying Tournament Suite under your own brand, [contact us](https://tournamentsuite.com/contact).

## Community Standards

All interactions in this repository are governed by our [Code of Conduct](CODE_OF_CONDUCT.md). We expect respectful, constructive engagement from everyone.

## Security Vulnerabilities

Please do **not** report security vulnerabilities as public issues. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
